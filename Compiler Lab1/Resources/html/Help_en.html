Text Editor - User's Guide

The "File" menu
To create opens a new empty file
Open opens a file on your computer in the .txt format
Save if the edited file has a save location on the computer, the edited file is saved with all the changes instead of the old one. If the file is saved for the first time, a dialog box opens, prompting the user to save the file in any location in the .txt format
Save as opens a dialog box that prompts the user to save the edited file in a convenient location
Exit exiting the app
The Edit menu
Undo the user's last action is canceled Ctrl+Z
Redo restoring a previously canceled user action Ctrl+Y
Cut saving the selected text to the clipboard and then deleting it Ctrl+X
Copy saving the selected text to the clipboard Ctrl+C
Paste entering the text saved in the clipboard into the input field Ctrl+V
Remove deleting selected text Del
Select all highlights all the text in the input field Ctrl+A
