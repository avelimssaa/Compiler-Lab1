Цикл for в языке Scala – это блок кода, который выполняется до тех пор, пока идентификатор в заголовке цикла не достигнет конечного значения путем прибавления к идентификатору единицы на каждой итерации цикла. В заголовке цикла задается начальное и конечное значение переменной, начальное и конечное значение переменной представляют из себя целое число без знака. В блоке кода, в рамках курсовой работы, используется оператор println(...), который выводит значение идентификатора на каждой итерации цикла.

Для объявления начала цикла используется ключевое слово for.

Формат записи: “for (идентификатор <- ЦБЗ to ЦБЗ) { println(идентификатор) };”.

В связи с разработанной автоматной грамматикой G[Z] синтаксический анализатор (парсер) будет считать верными следующие конструкции:

1. for (i <- 1 to 9) { println(i) };
2. for (g <- 0 to 13) {
    println(g)
    };
