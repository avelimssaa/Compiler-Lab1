Text Editor - User's Guide

The "File" menu
Alt+F
Create opens a new empty file Ctrl+N
Open opens a file on your computer in the .txt format Ctrl+O
Save if the edited file has a save location on the computer, the edited file is saved with all the changes instead of the old one. If the file is saved for the first time, a dialog box opens, prompting the user to save the file in any location in the .txt format Ctrl+S
Save as opens a dialog box that prompts the user to save the edited file in a convenient location Ctrl+Shift+S
Close the tab closes the current tab Ctrl+W
Exit exiting the app
